## 1.项目功能
- 本项目PSP(PicSearchPic)是带GUI的以图搜图工具
- 目前计划开发网络图片搜索与本地图片搜索两个功能

## 2.项目结构
```
.
├─ input                 # 本地图片文件夹[测试用]
├─ search                # 测试图片文件夹[测试用]
├─ PSP                   # 本项目主要功能的代码
│  ├─ __init__.py        # 初始化文件
│  ├─ read_pic.py        # 读取图片功能
│  ├─ search_pic.py          # 图片搜索功能
│  ├─ util.py            # 工具
├─ __init__.py           # 初始化文件
├─ main.py               # 主程序入口(运行此文件以启动)
```
#### 测试输入图片input的目录结构:

